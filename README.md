Proyecto de  de mini photoshop 

Funciones disponibles

El editor cuenta con las siguientes herramientas:

1. Escala de grises:  Convierte una imagen a blanco y negro manteniendo el formato RGB para compatibilidad con los demás filtros.
2. Rotación: Permite girar una imagen indicando el ángulo deseado. Ejemplo:  Ingrese ángulo: 90
3. Blur (Desenfoque): Aplica un efecto de desenfoque gaussiano. Permite controlar la intensidad mediante un radio. Ejemplo: Radio blur: 5
4. Ajuste de brillo: Modifica la iluminación de la imagen, ejemplo: Factor brillo:1.2

Valores:

Menor a 1 → oscurece
Mayor a 1 → aumenta luminosidad.

5. Ajuste de contraste: Permite aumentar o disminuir la diferencia entre zonas claras y oscuras. Ejemplo:  Factor  contraste:  1.3

6. Espejo horizontal:  Invierte la imagen horizontalmente.

7. Eliminación de fondo:  Utiliza inteligencia artificial mediante rembg para separar el objeto/persona del fondo.
Salida:  Imagen PNG con transparencia.
8. Fondo borroso: Genera un efecto tipo fotografía profesional: Fondo desenfocado.Persona u objeto principal enfocado.
9. Autoedición con IA: Aplica automáticamente una mejora general:

Aumento de brillo.
Mejora de contraste.
Incremento de color.
Mayor nitidez.

Pensado como una función similar a los filtros automáticos presentes en aplicaciones móviles.

10. Reset: Restaura la imagen al estado original cargado.
